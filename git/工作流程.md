## git工作流程相关内容

1. 如何找到一个bug是由谁提交的代码引起的呢？

   使用`git blame`，显示文件的每一行最后修改的版本和作者（既然会导致错误，那么现在也就是最后提交的时候肯定是有错误的，所以不需要考虑之前的修改）

   ```
   git blame fileName
   ```

   返回格式为：

   ```
   commit ID | 代码提交作者 | 提交时间 | 代码位于文件中的行数 | 实际代码
   ```

   对于commit ID，使用show命令展示它所指向的commit提交和所指向提交与上一次提交之间的差异。

   ```
   git show commitID
   ```
