## 使用

### app.config.js

```ts
export default {
  plugins: {
    myPlugin: {
      version: '1.0.0',
      provider: 'wxidxxxxxxxxxxxxxxxx'
    }
  }
}
```

声明版本和需要使用的插件

### page.config.js

```ts
export default {
  // 定义需要引入的插件
  usingComponents: {
    'hello-component': 'plugin://myPlugin/hello-component'
  }
}
```

### js 接口

使用插件的 `js` 接口时，可以使用 `Taro.requirePlugin` 方法。
